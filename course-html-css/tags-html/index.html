HTML : lenguaje de marcas de hipertexto

¿ Que es ?

HTML es un lenguaje utilizado para definir el contenido de las páginas web mediante un conjunto de etiquetas. Inicialmente diseñado para la divulgación de información con texto e imágenes, su propósito ha evolucionado con el tiempo, adaptándose a la creación de sitios web multimedia. Aunque originalmente no fue planificado para su uso tan amplio, ha recibido actualizaciones, y la versión más reciente es HTML5. Es un lenguaje fácil de aprender, lo que permite a cualquier persona, incluso sin experiencia en programación, crear páginas web básicas. A medida que se domina HTML, se pueden lograr diseños profesionales combinándolo con otros lenguajes como CSS. El código HTML se escribe en un archivo de texto con extensión .html o .htm, utilizando un editor de texto. Las etiquetas HTML indican cómo debe estructurarse el contenido, como párrafos, negritas o enlaces, y la mayoría de ellas requieren una etiqueta de cierre para marcar el fin de su efecto.

html (leguaje de marcas de hipertexto del ingles hipertext markup language) es el componente mas basico de la web define el significado y la estructura de la web.
